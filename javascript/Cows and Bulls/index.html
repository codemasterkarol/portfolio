Gameplay Instructions

Objective: Guess the randomly generated 4-digit number correctly.

Gameplay: The computer will generate a secret 4-digit number with each digit picked at random and ranging from 0-9, without repetition.

The player will then attempt to guess this 4-digit secret number. After each guess, the computer will compare the guess to the secret number and return a combination of Cows or Bulls (or none if you're unlucky).

Cow = number is found in secret number, but is not in the correct position.
Bull = number is found in secret number, and is in the correct position.
